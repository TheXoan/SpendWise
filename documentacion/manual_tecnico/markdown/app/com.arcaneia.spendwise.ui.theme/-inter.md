# Inter

[androidJvm]\
val [Inter](-inter.md): [FontFamily](https://developer.android.com/reference/kotlin/androidx/compose/ui/text/font/FontFamily.html)

Familia tipográfica principal utilizada en toda la aplicación.

Se basa en la fuente **Inter**, cargada manualmente desde `res/font`. Se definen tres pesos:

- 
   Normal
- 
   Medium
- 
   Bold

Esta familia es reutilizada en todos los estilos de texto de la app.
